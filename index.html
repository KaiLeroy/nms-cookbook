Welcome to NMS Comprehensible Cooking Guide!

Tutorial:
1.Choose the desired result in the input box.
2. Press "Lock In" to display all the recipe's possible ingredients.
3. Sometimes, there are multiple choices of ingredients for the same recipe. If this happens, --multiple choice-- will be visible and you can clear that input box to get possible ingredients list.
4. Press "Lock In" again when happy with your decision.
5. Keep doing that until you have nothing else to lock in. When it happens, you'll get info about all the base ingredients you need.
